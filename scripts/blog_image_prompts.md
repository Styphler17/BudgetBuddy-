# SpendScribe Blog — Featured Image Generation Prompts

**Style guide for all images:**
Use these settings consistently across all 10 images for a cohesive blog aesthetic.
- **Aspect ratio:** 16:9 (1280×720px minimum, 1920×1080 ideal)
- **Style:** Clean, modern, editorial photography aesthetic. Soft natural lighting. Muted, warm tones with subtle navy/blue accents to match SpendScribe brand colors.
- **Mood:** Calm, focused, optimistic — not stressful or chaotic.
- **Text/UI:** No text overlays, no phone screens, no app interfaces.
- **People:** If included, lifestyle/candid style. Avoid stock-photo poses. Partial shots (hands, desk, overhead) work better than full-face portraits.
- **Tool suggestions:** Midjourney, DALL-E 3, Adobe Firefly, or Stable Diffusion XL.

## Usage Notes

- **File naming:** Save images exactly as listed in the filename and place in `/public/blog/`
- **Format:** WebP only — all paths in the database use `.webp` extension
- **Compression:** Target under 200KB per image. Use [Squoosh](https://squoosh.app/) or [TinyPNG](https://tinypng.com/)
- **Alt text:** Already set in the upgrade SQL for posts 1–3 and 5 — no changes needed after upload
